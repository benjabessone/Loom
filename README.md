# Loom Digital - Landing Page

## Descripción

Sitio web profesional para Loom Digital, especialistas en creación de landing pages de alta conversión. El proyecto incluye una página principal y páginas especializadas para cada servicio.

## Estructura del Proyecto

```
loom-clean/
├── index.html              # Página principal
├── css/
│   └── styles.css          # Estilos principales optimizados
├── js/
│   └── script.js           # JavaScript refactorizado y optimizado
├── images/
│   ├── Logo.png            # Logo principal
│   ├── hero-bg.jpg         # Imagen de fondo del hero
│   └── ...                 # Otras imágenes
├── especialidades/
│   ├── diseno-personalizado.html
│   ├── optimizacion-cro.html
│   ├── desarrollo-responsive.html
│   └── analisis-seguimiento.html
├── sitemap.xml             # Mapa del sitio para SEO
├── robots.txt              # Directivas para crawlers
└── server.js               # Servidor de desarrollo
```

## Instalación y Uso

### Desarrollo Local

1. Clona o descarga el proyecto
2. Navega al directorio del proyecto
3. Ejecuta el servidor de desarrollo:

```bash
node server.js
```

4. Abre tu navegador en `http://localhost:3000`

### Producción

El proyecto está listo para ser desplegado en cualquier servidor web estático. Simplemente sube todos los archivos al directorio raíz de tu hosting.
